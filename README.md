# Portfolio 💻

Portfolio professionnel responsive développé afin de présenter mes projets, compétences et expériences en développement web front-end.

Ce portfolio a été conçu comme une véritable vitrine professionnelle mettant en avant mes réalisations, mes compétences techniques ainsi que mon approche du développement moderne et de l’expérience utilisateur.

## Features

- Design responsive mobile, tablette et desktop
- Présentation dynamique des projets
- Interface moderne et animations interactives
- Section compétences et technologies
- Formulaire de contact
- Téléchargement du CV
- Architecture de composants réutilisables
- Optimisation des performances
- Navigation fluide
- Support multilingue avec i18n

## Tech Stack

- React
- JavaScript
- HTML5
- SCSS / CSS3
- Vite
- React Router
- i18next
- Framer Motion
- React Icons
- Git / GitHub
- Github Pages
